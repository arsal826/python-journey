# My Python Journey

A structured, week-by-week record of my Python learning — from basics to building APIs with FastAPI.

## Structure

| Folder | Topics |
|--------|--------|
| `week-1/` | Variables, data types, conditionals, loops, string operations |
| `week-2/` | String manipulation, lists, dictionaries, data grouping |
| `week-5/` | Functions, built-in functions, list operations, algorithms |
| `week-6/` | Complex data structures, nested dicts/lists, data analysis |
| `api-journey/` | Building REST APIs with FastAPI |

Each week is organized into daily folders (`day-1/`, `day-2/`, ...) containing practice problems.

## Tech Stack

- **Language:** Python 3.14
- **API Framework:** FastAPI
- **Formatter:** Black

## How to Run

```bash
# Run any practice file
python week-1/day-1/day1.py

# Run the API (requires FastAPI + Uvicorn)
cd api-journey/day-1
uvicorn api:app --reload
```
